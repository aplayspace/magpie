# Magpie - Redaction Poetry Web App

A mobile-optimized web app for creating erasure poetry through text redaction.

## Features

- **Interactive Word Redaction**: Tap any word to black it out
- **Undo Capability**: Tap redacted words again to reveal them
- **Random Text Loading**: Get new content with the "New Text" button
- **Export & Share**: Screenshot your creation and share on social media
- **Mobile-First Design**: Optimized for phone and tablet use

## How to Use

1. Open `index.html` in a web browser
2. Tap words to redact them (they'll turn black)
3. Tap redacted words again to un-redact them
4. Click "New Text" to load different content
5. Click "Export" to screenshot and share your creation

## Sharing the MVP

### Option 1: Local Testing
Simply open the `index.html` file in any modern web browser.

### Option 2: Deploy to GitHub Pages
1. Create a new GitHub repository
2. Push these files to the repo
3. Enable GitHub Pages in repository settings
4. Share the URL: `https://yourusername.github.io/magpie`

### Option 3: Deploy to Netlify/Vercel
1. Drag and drop this folder to Netlify Drop or Vercel
2. Get an instant public URL to share
